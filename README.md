# Archipelago Manual - Sonic Speed Simulator

## Goals
- Defeat Dr. Eggman (Normal or Hard)
- Defeat Neo Metal Sonic (Normal or Super)
- Reach the Final Zone (currently Metro City)
- Complete All Main Quests

## Items
- Progressive World Keys
- Progression Characters (used to reach Zones or fight Bosses)
- Other Playable Characters

## Checks
- Reaching a Zone for the first time
- Completing Quests
- Defeating Bosses
